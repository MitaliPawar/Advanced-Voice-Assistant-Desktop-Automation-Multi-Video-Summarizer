# Advanced-Voice-Assistant-Desktop-Automation-Multi-Video-Summarizer
A smart AI-powered desktop voice assistant built using Python.
It performs hands-free desktop automation, opens applications, writes text, performs Google/Wikipedia searches, and can generate summaries of multiple YouTube videos.
The project includes a clean HTML + CSS UI (via Flask) to display results and summaries in a user-friendly interface.
